# 🏦 Bank Manager

A simple Java console application that performs basic banking operations using Object-Oriented Programming.

## Features

* Create a bank account
* Display all accounts
* Search an account by account number
* Deposit money
* Withdraw money
* Menu-driven interface
* Exception handling for invalid input
* Save account data to a file
* Load saved accounts when the program starts

## Technologies Used

* Java
* OOP (Classes & Objects)
* ArrayList
* Exception Handling
* File Handling

## Project Structure

Bank Manager

## How to Run

1. Clone the repository.
2. Open the project in VS Code or any Java IDE.
3. Run `Main.java`.
